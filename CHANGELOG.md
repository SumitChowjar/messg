# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.2.1] - 2025-11-18

### Added
- Added demo.html for browser testing and development
- Added comprehensive test suite for all message types and features
- Added TypeScript definitions for better IDE support
- Added .npmignore for explicit npm package control
- Added prepublishOnly script for safety checks before publishing

### Changed
- Improved package.json configuration for better npm publishing
- Enhanced documentation with more examples

### Fixed
- Improved code quality and linting standards

## [2.2.0] - Previous Release

### Added
- Initial maintained fork of the original messg package
- Full ES6 module support
- Responsive CSS with media queries
- Support for multiple message positions
- Message flow management with max limit
- Global options for speed, position, and flow control

### Features
- Default, success, info, warning, and error message types
- Button support with custom handlers
- Hide handlers for message lifecycle management
- Auto-hide with configurable delay
- Message stacking and flow control
- Responsive design for mobile and desktop

## [1.0.0] - Original Release

### Initial Release
- Original messg package by Andrey Polischuk
- Basic message notification system
- CSS3 animations
- Multiple message types
